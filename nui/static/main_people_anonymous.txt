            <h2>Become a member</h2>
            <p>Our tools help you <a style="" href="http://www.openplans.org/projects/openplans-help/create-a-new-project">create</a> or <a style="" href="http://www.openplans.org/projects/openplans-help/contribute-to-an-existing-project">contribute</a> to web sites, <a style="" href="http://www.openplans.org/projects/openplans-help/add-files-to-a-page">share files</a>, upload photos, <a style="" href="http://www.openplans.org/projects/openplans-help/create-and-manage-mailing-lists">manage mailing lists</a>, and manage task lists. OpenPlans is collaborative software so most <a style="" href="http://www.openplans.org/projects/openplans-help/edit-an-existing-page">pages are editable</a>. It takes just seconds to <a style="" href="http://www.openplans.org/join_form">register</a> and, once you <a style="" href="http://www.openplans.org/login_form">log in</a>, you can <a style="" href="http://www.openplans.org/projects/openplans-help/create-a-new-project">start your own project</a> or <a style="" href="http://www.openplans.org/projects/openplans-help/contribute-to-an-existing-project">contribute to an existing one</a>.</p>
